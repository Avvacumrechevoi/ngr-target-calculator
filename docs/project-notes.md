# Project notes

Этот проект собран как статическое приложение без сборщика и без npm.
React, ReactDOM и Babel подключаются через CDN прямо в `index.html`.

Это значит:
- не нужен `package.json`
- не нужен `vite` или `webpack`
- проект можно сразу загрузить в GitHub и опубликовать через GitHub Pages
