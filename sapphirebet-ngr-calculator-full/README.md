# SapphireBet NGR Calculator

Готовый статический проект калькулятора NGR для iGaming.

## Структура
- `index.html` — основной файл приложения
- `src/original/ngr-calculator-source.html` — исходный загруженный файл
- `docs/project-notes.md` — краткие заметки по проекту
- `.gitignore`
- `LICENSE`

## Как загрузить в GitHub
1. Создай новый репозиторий.
2. Загрузи в него все файлы из этой папки.
3. Для публикации открой **Settings → Pages**.
4. В разделе **Build and deployment** выбери:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Сохрани настройки — сайт опубликуется через GitHub Pages.

## Локальный запуск
Открой `index.html` в браузере.
